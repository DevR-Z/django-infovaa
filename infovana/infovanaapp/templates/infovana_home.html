Infovana App

Bienvenido {{user.username}}

Infovana App es una aplicación web para la recolección de datos de la API OMDB (la cual es una API de películas) a través de peticiones request, con la finalidad de que el usuario pueda ver el detalle de películas o series y añadirlas como favoritas

Tecnologías usadas:

-Lenguaje de Programación: Python

-Framework: Django

-Base de Datos: SQLite

-Frontend: HTML, CSS, Bootstrap

-Autenticación: Django Authentication

-API Externa: OMDB API (usando la librería `requests`)
